# OmniHub POS Portal v1.0.0

Live URL after GitHub Pages deployment:

`https://georgebundo.github.io/OmniHub-POS-Portal/`

## Publishing steps

1. Create a new **public** repository named `OmniHub-POS-Portal`.
2. Publish these files to its `main` branch.
3. Open **Settings → Pages**.
4. Choose **Deploy from a branch**.
5. Select `main` and `/(root)`.

The repository contains only browser frontend files and a public Supabase
publishable key. It contains no database password, service-role key, secret
project key, personal access token, or administrator bootstrap token.
